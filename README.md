# Career Pathways

##  Project Description
Developed a web-based Career Pathways System to help students explore suitable career options based on their interests and skills.The application provides structured guidance on different career paths and improves decision-making for students.

Built the frontend using HTML, CSS, and JavaScript with a responsive user interface. This project enhanced my skills in web development, user interface design, and logical thinking.

##  Features
- Career guidance pages
- Courses information
- Government jobs section
- Interview preparation
- Responsive design

##  Technologies Used
- HTML
- CSS
- JavaScript
